# Secrets Directory

This directory is intentionally used for local secret files generated by `make`.

Tracked files:

- `README.md`

Ignored files created locally:

- `db_root_password.txt`
- `db_password.txt`
- `wp_admin_password.txt`
- `wp_user_password.txt`
- `credentials.txt`

Do not commit the generated `.txt` files.
